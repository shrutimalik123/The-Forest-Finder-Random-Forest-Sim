# 🌲 The Forest Finder: Random Forest Ensemble Sim

An interactive Machine Learning simulation designed to teach the power of **Ensemble Learning**. You play as an Environmental Scientist building a "Random Forest" to predict wildfire risks. By combining multiple "weak" Decision Trees, you will learn how "Majority Voting" creates a model that is significantly more accurate and stable than its individual parts.

## 🎓 Learning Objectives

This project focuses on teaching:
* **Decision Trees:** Understanding the basic "if-then" logic used to split data based on features like temperature and humidity[span_0](start_span)[span_0](end_span)[span_1](start_span)[span_1](end_span).
* **Ensemble Learning:** The strategy of using multiple models together to solve a complex problem[span_2](start_span)[span_2](end_span)[span_3](start_span)[span_3](end_span).
* **Bagging (Bootstrapping):** How each tree is trained on different slices of data to ensure diversity[span_4](start_span)[span_4](end_span).
* **Majority Voting:** The process where the forest's final prediction is determined by the most common output of its individual trees[span_5](start_span)[span_5](end_span)[span_6](start_span)[span_6](end_span).
* **Variance Reduction:** Learning how a forest prevents the "overfitting" common in single decision trees[span_7](start_span)[span_7](end_span).

---

## ✨ Features

* **Wildfire Prediction Scenario:** A high-stakes environmental use case that makes abstract logic easy to visualize.
* **Forest Scaling:** Choose the number of trees to see how a larger ensemble stabilizes the prediction.
* **Tree-Level Logic:** Watch the internal "thinking" of every tree as it sets its own randomized thresholds.
* **Democratic Inference:** Simulates the majority voting process to reach a final classification.
